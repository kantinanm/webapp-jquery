# How to install

Use powershell or cmd and type by order, please see below.

- `git clone https://github.com/kantinanm/webapp-jquery.git`
- `cd webapp-jquery`
- > install package dependency in this project.
  - `npm install`

# How to test local

- > run main js file.
  - `npm run serve`
